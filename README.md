
🧮 Simple Calculator

A responsive calculator built using HTML, CSS, and JavaScript.
All JavaScript logic is written inside the HTML file using the <script> tag.

🚀 Features

➕ Addition
➖ Subtraction
✖ Multiplication
➗ Division
🧹 Clear Display
📱 Responsive Design
⚡ Real-time Calculation

🛠 Technologies Used

HTML – Structure and layout
CSS – Styling and design
JavaScript (Internal Script) – Calculation logic and DOM manipulation

📂 Project Structure
Calculator/
│── index.html
│── style.css

JavaScript code is included inside the index.html file.

💡 Concepts Used

Variables
Functions
Event Handling
DOM Manipulation
querySelector()
String Operations

▶ How to Run

Download or clone the repository
Open index.html in your browser
Start calculating 🎉

🎯 Purpose

This project was created to practice basic JavaScript and DOM manipulation concepts.
